Простір для релаксу

Наш просторий відкритий басейн — ідеальне місце для літнього відпочинку. Оточений зеленню парку, він створює атмосферу справжнього курорту посеред мальовничої Львівщини.

Зона відпочинку обладнана зручними шезлонгами та парасольками для максимального комфорту гостей.

Для всієї родини

Відпочинок для кожного

Басейн підходить як для дорослих, так і для дітей. Чиста вода, безпечна зона та доглянута територія — все для того, щоб ваш відпочинок був бездоганним.

Поруч розташований дитячий парк розваг, де найменші гості зможуть весело провести час, поки дорослі насолоджуються спокоєм.
